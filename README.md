
# Crowd Funding Web




## Tech Stack

**Front End:** Vue JS, Nuxt JS

**Back End:** Node, Golang



## Run Locally

Clone the project

```bash
  git clone https://github.com/ikhwankhaliddd/Crowd-Funding-Project.git
```

Go to the project directory

```bash
  cd Crowd-Funding-Project-master
```

Install dependencies

```bash
  go mod tidy
```

Start the server

```bash
  go run main.go
```
